XML-Sitemap Generator
===============================

Bash script for generating XML-Sitemaps used by searchengine crawlers.

__How to use__

    ./xmlsitemap.sh <file> <url>

    ./xmlsitemap.sh urls.txt http://www.example.com   

__Options__

    <file> File with one url per line.
    <url> base url
